Vector Space

What is a vector space?
Recall that we've defined a vector as a quantity with a magnitude and a direction.
This is an informal definition however, and can only take us so far.
Vector spaces give us a more generalized definition of a vector.
Like just about everything in math, generalization helps give a broader understanding of a topic as a whole. Anything we prove about vector spaces can be applied to anything which can be described as a vector space.

Requirements

For a valid vector space, we need a few things, namely:

  • A scalar field.
  • Elements for the vector space. (The actual vectors)
  • Definition of vector addition and scalar multiplication.


We can choose to fill in these requirements with whatever we want as long as it satisfies a few specific conditions/axioms which will be simplified for now.

Scalar Field

A field can be described as a set of elements where addition, subtraction, multiplication, and division are defined to produce another element in the set.
(Subtraction and division can be omitted from the definition here as they are special cases of multiplication and addition, just like their vector counterparts)
We need to pick a scalar field to use for the scalar numbers.
The scalar field we've been using so far is the real number field, which contains every 'ordinary' number like 3, -5, 2.2, \(\pi\), etc. Using this field creates a real vector space.
Note that a vector space is not a field because, as stated earlier, there is no clear multiplication between two vectors.

Elements for the vector space

These are the actual elements which are considered the 'vectors' of the vector space.
They can be added together or multiplied by an element of the scalar field.
Remember, anything which satisfies those properties works here.
If you managed to define vector addition and scalar multiplication for a rock, it could be considered a vector.
This gives us a more abstract definition of a vector, which would be any element in which vector addition and scalar multiplication can be applied to.

Defining Vector Addition and Scalar Multiplication

Lastly, we need to define what vector addition and scalar multiplication will do.
In the previous section, we've already seen the standard definition of vector addition and scalar multiplication, though anything works as long as we adhere to some axioms (like the commutativity and associativity rules for addition to name a couple).
These are the called the vector space axioms, though we won't concern ourselvese too much with them.
One important thing to note though, is that the result of the operations must also be an element of the vector space.

Vector Space Axioms

Note that there are multiple ways to phrase these axioms, but they should all be equivalent.

Real Vector Space

As stated earlier, the type of vector space we're familiar with is the real vector space.
For the requirements:

  • The scalar field is the set of real numbers.
  • Elements have a magnitude and a direction.
  • Definitions of vector addition and scalar multiplication are from the previous section.


For other types of vector spaces, our older definition of a vector breaks down here, as magnitude and direction is not always assumed to exist.

Summary

A vector space generalizes vectors.
There are three things we need for a vector space, a scalar field, the actual vectors, and definitions for vector addition and scalar multiplication.
A field can be described as a set where addition and multiplication produces another element also in the set.
Vectors can be redefined as any element in which vector addition and scalar multiplication can be applied to.
Vector addition and scalar multiplication must satisfy the vector space axioms.
